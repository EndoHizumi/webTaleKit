# 画像の差し替え

プログラミングの知識がなくても、画像ファイルを差し替えるだけで簡単にゲームをカスタマイズできます。

## 基本的な手順

既存の画像ファイルを新しい画像で置き換えます。**ファイル名は同じにしてください。**

::: warning 注意
ファイル名を変更すると、ゲームが正しく動作しなくなります。必ず元のファイル名と同じ名前で保存してください。
:::

## キャラクター画像を変える

キャラクター画像は `./src/resource/chara/` ディレクトリにあります。

例えば、ガイドキャラクターを変更する場合：

1. 新しいキャラクター画像を用意する（PNG形式推奨）
2. `./src/resource/chara/guide.png` を新しい画像で上書きする
3. ファイル名は必ず `guide.png` のままにする

## 背景画像を変える

背景画像は `./src/resource/background/` ディレクトリにあります。

例えば、タイトル画面の背景を変更する場合：

1. 新しい背景画像を用意する
2. `./src/resource/background/title_bg.png` を新しい画像で上書きする
3. ファイル名は必ず `title_bg.png` のままにする

## BGMを変える

BGMは `./src/resource/bgm/` ディレクトリにあります。

例えば、タイトル曲を変更する場合：

1. 新しい音楽ファイルを用意する（MP3形式推奨）
2. `./src/resource/bgm/title_theme.mp3` を新しいファイルで上書きする
3. ファイル名は必ず `title_theme.mp3` のままにする

## ボタンの見た目を変える

選択肢ボタンの画像は `./src/resource/system/systemPicture/02_button/` ディレクトリにあります。

ボタンには3つの状態があります：

- **button.png** - 通常時（何もしていない状態）
- **button2.png** - マウスを乗せた時（ホバー状態）
- **button3.png** - クリック時（選択状態）

それぞれの画像を新しい画像で上書きして、ボタンの見た目を変更できます。

## パス表記について

::: info パスの記法
- `./` は「現在のプロジェクトフォルダから」という意味です
- パス区切り文字は `/` (スラッシュ) を使用しています
- **Windowsをお使いの方:** `\` (バックスラッシュ) でも動作しますが、`/` 形式を推奨します
:::

## 推奨画像形式

| リソースタイプ | 推奨形式 | 備考 |
|--------------|---------|------|
| キャラクター | PNG | 透過背景が使える |
| 背景 | JPG, PNG | JPGの方がファイルサイズが小さい |
| ボタン | PNG | 透過背景が使える |
| BGM | MP3, OGG | MP3が一般的 |
| SE | MP3, OGG | 短い音は軽量なOGGも良い |
